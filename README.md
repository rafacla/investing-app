# budget-app
## English
This project aims in build a web app alternative to a budgeting tool (like YNAB but better I hope). 
It's built using CodeIgniter, Bootstrap and heavy use of jquery to interface.

It will be integrated with an investment tool (brazilian stock market and funds).

The budget app supports english in about 90% of it (change default language in codeigniter config file).

## Portuguese
Este projeto foca em escrever um web app como alternativa para ferramentas de controle de despesas (como o YNAB, mas melhor).
É escrito em cima de CodeIgniter, Bootstrap e bastante uso de jquery para otimizar a interface.

Será integrado com uma ferramenta de investimentos (mercado de ações brasileiro e fundos para renda fixa).

A parte do APP de orçamentos tem suporte a traduções (inglês está pronto em 90% do app).
